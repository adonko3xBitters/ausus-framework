# Changelog — ausus/presentation-default

## [0.1.0] — 2026-05-19

**Name reservation release.** No source code shipped.

This package reserves the `ausus/presentation-default` name on Packagist
while the dedicated L5 Presentation + L3 ReportingDriver + standard
field-types package (RFC-004, RFC-010, RFC-012 §7/§8) is being finalized.
For V0, the `ProjectionRenderer` is shipped inline by
`ausus/runtime-default` — see that package's CHANGELOG.

Consumers should not require this package until v0.2.0.
